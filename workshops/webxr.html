WebXR Development

Building for the Immersive Web

Workshop Overview

A seminar exploring the capabilities of the WebXR Device API. This talk covered the current state of browser-based VR/AR, optimizing 3D assets for the web using glTF, and deploying experiences using Three.js and A-Frame.

Topics Covered

  • Framework Comparison

    Three.js vs A-Frame vs Babylon.js - choosing the right tool for the job.

  • Performance Optimization

    Techniques for maintaining 90fps in the browser, including geometry instancing and texture compression.
